# DO₂計算機

酸素供給量（Oxygen Delivery）を計算するWebアプリケーション

## 概要

血液ガス分析データから酸素供給量（DO₂）を計算する医療計算ツールです。

## 機能

- **入力項目**
  - ヘモグロビン (Hb): 3-18 g/dL
  - 動脈血酸素飽和度 (SaO₂): 60-100%
  - 心拍出量 (CO): 0-10 L/min

- **計算式**
  ```
  CaO₂ = Hb × 1.34 × (SaO₂ / 100)
  DO₂ = CO × CaO₂ × 10
  ```

- **アラート機能**
  - DO₂ ≤ 400 mL/min: 🚨 レッドアラート（重度の酸素供給不足）
  - 400 < DO₂ ≤ 500 mL/min: ⚠️ イエローアラート（酸素供給低下）
  - DO₂ > 500 mL/min: ✓ 正常範囲

## 使い方

1. ブラウザで `index.html` を開く
2. スライドバーまたは数値入力で値を入力
3. リアルタイムでDO₂が計算されます

## 特徴

- レスポンシブデザイン（モバイル対応）
- リアルタイム計算
- スライドバーと数値入力の両方に対応
- オフラインで動作
